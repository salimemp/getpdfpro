# How to Merge PDFs: The Complete Guide (2026)

Need to combine multiple PDF files into one document? Maybe you're assembling
a job application packet, bundling invoices for your accountant, putting
together a research paper, or sending several contracts in one email. Whatever
the reason, merging PDFs is one of the most common document tasks — and one
of the easiest to get wrong if you don't know the tricks.

This guide covers every way to merge PDFs in 2026, on every device, with
clear step-by-step instructions. No fluff, no upsells, no "but first, sign
up for our newsletter." Just the steps.

---

## What you'll learn

- How to merge PDFs online for free (no software install)
- How to merge PDFs on Windows, macOS, Linux, iPhone, and Android
- How to merge PDFs offline with desktop software
- How to merge password-protected PDFs
- How to merge specific pages from different PDFs
- How to reorder pages after merging
- Common mistakes and how to avoid them
- Quick fixes when things go wrong

**Time to complete:** 2–5 minutes depending on the method.

## Before you start: a quick checklist

✅ All your files are PDFs (not Word, JPG, or other formats)
✅ You have at least 2 files to combine
✅ Your files aren't total bigger than 100 MB (free tier) or 4 GB (Pro)
✅ You have permission to combine the documents (especially for copyrighted material)

> **Tip:** If you have JPGs or Word docs you want in the merged file, convert
> them to PDF first. See [how to convert JPG to PDF](/blog/jpg-to-pdf) and
> [how to convert Word to PDF](/blog/word-to-pdf).

## Method 2: Merge PDFs on Windows

Windows 10 and 11 don't have a built-in PDF merger, but you have options:

### Option A: Microsoft Edge browser
1. Open one of the PDFs in Microsoft Edge
2. Click the **"..."** menu → **"Open with Edge"** if not already
3. Press **Ctrl+P** to print
4. Change the printer to **"Microsoft Print to PDF"**
5. Click **"Print"**
6. **This only works for one file at a time** — for actual merging, use Option B

### Option B: GetPDFPro web tool
Same as Method 1. Works identically on Windows.

### Option C: Adobe Acrobat Reader
1. Open Adobe Acrobat Reader
2. **Tools** → **Combine Files**
3. Click **"Add Files"** and select your PDFs
4. Rearrange the order
5. Click **"Combine"**
6. Save the merged file

> **Note:** Adobe Acrobat Reader is free, but the **Combine Files** tool is
> only available in **Acrobat Pro** (paid). Reader users will see a prompt
> to upgrade.

### Option D: Third-party desktop software
- **PDFsam Basic** (free, open source)
- **PDF XChange Editor** (free tier available)
- **Foxit PDF Reader** (free tier available)

These are all good for offline merging, especially if you handle sensitive
documents you don't want to upload anywhere.

## Method 3: Merge PDFs on macOS

macOS has a built-in PDF merger in **Preview** — no extra software needed.

### Step-by-step with Preview
1. Open one of the PDFs in **Preview** (the default Mac PDF viewer)
2. Click **View** → **Thumbnails** (or press **Option+Cmd+2**)
3. A sidebar will appear showing each page as a thumbnail
4. Drag another PDF file from Finder **into the sidebar**
5. The new file's pages will be added to the end
6. To rearrange, drag thumbnails up or down
7. **File** → **Export as PDF** to save the merged version

> **Tip:** This is the cleanest, most private way to merge PDFs on a Mac.
> Nothing leaves your computer.

### Alternative: GetPDFPro web tool
Same as Method 1. macOS works perfectly in any browser.

### Alternative: Automator
For power users, you can create an Automator workflow that merges PDFs from
a folder. Search "PDF" in Automator's actions panel.

## How to merge specific pages from different PDFs

Sometimes you don't want the whole file — just pages 2–4 from one and
pages 1–3 from another.

### Using GetPDFPro
1. Go to [getpdfpro.com/tools/merge](https://getpdfpro.com/tools/merge)
2. Upload all the files
3. After upload, **expand each file's thumbnail** (click the chevron)
4. **Type page ranges** like `1-3, 5, 8-10` to select specific pages
5. Reorder as needed
6. Click **"Merge"**

### Using Preview (Mac)
1. Open the first PDF in Preview
2. Show thumbnails (**View** → **Thumbnails**)
3. Drag the second PDF into the sidebar
4. To remove unwanted pages, select them and press **Delete**
5. Save (**Cmd+S**)

## How to merge password-protected PDFs

Password-protected PDFs are trickier because most tools can't read the
content without the password.

### If you know the password
1. Use [GetPDFPro's unlock tool](/tools/unlock) first to remove the password
2. Then merge as normal

### If you don't know the password
- You can't merge the file (and shouldn't try — it's protected for a reason)
- If you have permission, ask the sender for an unprotected copy

> **Security tip:** Never upload sensitive password-protected documents to
> random online tools. Use offline software or zero-knowledge mode.

## How to maintain quality when merging

The merge process should **never reduce quality** — the tool is just
combining files, not re-encoding. If you notice quality loss, something
else is happening.

### Common quality issues

**Issue: "My file size exploded after merging"**
This is normal — you're adding the contents of multiple files into one.
A 2 MB PDF + 3 MB PDF = 5 MB merged file. If your merged file is much
bigger than the sum, try [compressing it](/tools/compress).

**Issue: "The fonts look wrong in the merged file"**
Some PDFs use non-standard fonts that get substituted. This is rare with
modern tools. If it happens, use the original tool that created the PDFs
to re-export with embedded fonts.

**Issue: "Images look pixelated"**
The merge didn't change image quality. The original PDFs probably had
low-resolution images. Re-export from the source with higher quality
settings.

## Common mistakes and how to avoid them

### Mistake 1: Merging the wrong order
Always check the order of your files before clicking "Merge." Page 1 of
the first file becomes page 1 of the merged document.

### Mistake 2: Including blank pages
Some tools include trailing blank pages. Open your merged file and check
page 1 and the last page.

### Mistake 3: Exceeding file size limits
Free tiers usually cap at 100 MB. Compress your files first if needed.

### Mistake 4: Uploading sensitive data to random sites
For medical, legal, or financial documents, use offline software or
zero-knowledge mode.

### Mistake 5: Losing the original files
Always keep the source PDFs. If the merged file is wrong, you can
re-merge without re-downloading.

### Mistake 6: Not naming the merged file descriptively
A merged file called "document.pdf" is useless when you have five of them.
Use the [rename tool](/tools/rename) to give it a meaningful name.

## Troubleshooting

### "The merge button does nothing"
- Check your internet connection
- Make sure all uploads completed (look for green checkmarks)
- Try a different browser
- Disable browser extensions (ad blockers sometimes interfere)

### "I uploaded but the file order is wrong"
- Drag thumbnails to reorder
- Or remove all files and re-upload in the right order

### "The merged file won't open"
- File might be corrupted — try re-merging
- Try opening in a different PDF reader (Adobe, Preview, Foxit)
- Check if the file size is 0 KB (means merge failed)

### "I get an error about file size"
- Free tier is 100 MB total
- Upgrade to Pro for 4 GB
- Or split your files and merge in batches

### "Some pages are missing"
- They might be in a different PDF you didn't include
- Some PDFs have hidden layers — use the [organize tool](/tools/organize) to verify

### "It's taking forever"
- Big files take longer
- Free tier may queue jobs during peak hours
- Pro users get priority processing

## Frequently Asked Questions

### Is merging PDFs safe?
Yes, with reputable tools. GetPDFPro uses TLS encryption in transit, AES-256
at rest, and auto-deletes files after 24 hours. For maximum privacy, use
**Zero-Knowledge Mode** or offline software.

### Are there any file size limits?
- **GetPDFPro Free:** 100 MB total per merge
- **GetPDFPro Pro:** 4 GB total per merge
- **iLovePDF Free:** 100 MB (varies by tool)
- **Adobe Acrobat Pro:** No practical limit (uses your disk)

### Can I merge PDFs on my phone?
Yes. Use the [GetPDFPro iOS app](https://apps.apple.com/app/getpdfpro),
[Android app](https://play.google.com/store/apps/details?id=app.getpdfpro),
or the web tool in mobile Safari/Chrome.

### How many PDFs can I merge at once?
- **Free tier:** Up to 20 files in one merge
- **Pro tier:** Up to 100 files in one merge
- **Bulk (1000+):** Use the API or jobs endpoint

### Can I merge a PDF with a Word document?
First convert the Word document to PDF using
[Word to PDF conversion](/blog/word-to-pdf). Then merge.

### Will merging reduce the file quality?
No. The merge is a non-destructive operation — the tool combines files
without re-encoding. Quality stays exactly the same as the source files.

### Can I merge scanned PDFs?
Yes, but they're treated as image-based PDFs. If you want to extract text
from them first, run [OCR](/tools/ocr) on each file before merging.

### Does it work on Linux?
Yes. Use the web tool in any browser, or install offline tools like
PDFsam, LibreOffice Draw, or pdfunite.

### Will the merged PDF work in Adobe Acrobat?
Yes. Merged PDFs are standard PDF files compatible with all PDF readers
and editors.

### Can I undo a merge?
The merge itself can't be "undone," but you can
[split the merged file](/tools/split) back into individual files if
needed.

### Do I need to create an account?
For basic one-time merges, no. For larger files, batch processing, or
zero-knowledge mode, yes (free account works).

### What happens to my files after merging?
On GetPDFPro, uploaded files are auto-deleted after 24 hours. We don't
share them with anyone, and we don't train AI on them (unless you
explicitly use AI features, which are opt-in).
