# Internationalization (i18n / l10n)

## Goals
- 25 languages at launch (matching iLovePDF parity)
- 40+ languages by year 2
- RTL support (Arabic, Hebrew, Urdu, Persian)
- Locale-aware formatting (numbers, dates, paper sizes)
- Translated UI strings, AI prompts, help docs, error messages

## Locale Routing

Web uses `[locale]` segment:
- `getpdfpro.com/en/tools/merge`
- `getpdfpro.com/es/tools/merge`
- `getpdfpro.com/ar/tools/merge` (RTL)

Mobile uses device locale by default, with in-app switcher.

## AI Prompt Translation

The AI orchestrator injects locale instructions:
```python
lang_instruction = {
    "en": "Respond in English.",
    "es": "Responde en español.",
    "ar": "أجب باللغة العربية.",
    "hi": "हिंदी में जवाब दें।",
    # ... 25+ entries
}.get(language, f"Respond in the user's language ({language}).")
```

Gemini is strong at multilingual output, no separate translation step needed.

## OCR Languages

Tesseract OCR supports 100+ languages. We bundle the most common at install time:
- English, Spanish, French, German, Italian, Portuguese (Latin)
- Arabic, Hebrew (RTL)
- Hindi, Bengali, Tamil, Telugu (Indic)
- Chinese (Simplified + Traditional), Japanese, Korean (CJK)
- Russian, Ukrainian, Polish, Czech (Cyrillic + Slavic)

To add a new language:
1. Add `tesseract-ocr-{code}` to Dockerfile
2. Add language code to UI language picker
3. Re-deploy

## Translation Workflow

1. Source: English (`en.json`) — the canonical file
2. Translation: use **Crowdin** or **POEditor** (free for open-source, paid for private)
3. PR workflow: translation files auto-generate PRs when source changes
4. Review: native speakers volunteer or paid translators
5. Release: ship with whatever languages are ≥95% translated

## Locale-Aware Formatting

```dart
// Flutter example
import 'package:intl/intl.dart';

final dateFormat = DateFormat.yMMMMd(locale.toString()); // "January 6, 2026" or "6 जनवरी 2026"
final numberFormat = NumberFormat.decimalPattern(locale.toString()); // "1,234.56" or "1.234,56"
```

Paper sizes:
- US/Canada/Mexico: `Letter` (8.5" × 11")
- Everywhere else: `A4` (210mm × 297mm)
- Detected from user locale, overridable in settings

## RTL Support

- CSS: Tailwind `rtl:` variants
- Flutter: `Directionality` widget
- Layouts: use logical properties (`start`/`end` not `left`/`right`)
- Icons: mirror arrows, chevrons, etc.
- Test with Arabic locale from day 1

## Future: Voice in All Languages

The read-aloud feature uses:
- Web: Web Speech Synthesis API (depends on browser, system voices)
- Mobile: native TTS engines
  - iOS: 50+ voices, on-device neural since iOS 13
  - Android: 100+ voices via Google TTS

All languages supported by the platform's TTS work out of the box.
